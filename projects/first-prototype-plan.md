# First Prototype Plan

The first prototype should be simple and realistic.

The goal is not to build a perfect product immediately. The goal is to create something small that works and can be improved later.

## Best first project

The best first project for this repository is a text based booking assistant.

It can ask a few questions, collect answers and create a clear booking summary.

## Prototype flow

1. Ask for customer name
2. Ask for phone or email
3. Ask what service the customer wants
4. Ask for preferred day
5. Ask for preferred time
6. Ask for extra notes
7. Show a final summary
8. Save the information in a simple file or spreadsheet

## Why this is a good first step

This project is simple but useful.

It teaches how to collect information, organize data and prepare the base for a real automation system.

## Possible tools

1. Python
2. CSV file
3. Google Sheets later
4. Simple web form later
5. AI reply logic later
