# Roadmap

This roadmap shows the next steps for the AI Automation Lab repository.

## Stage 1

Set up the repository with clean documentation and clear project ideas.

Status: started

## Stage 2

Choose the first simple automation project.

Recommended choice: text based booking assistant

## Stage 3

Create the first Python prototype.

The prototype should ask questions, collect booking details and save the information.

## Stage 4

Improve the prototype with better structure.

Possible improvements include input validation, cleaner summaries and saving data in a better format.

## Stage 5

Connect the project with a spreadsheet or a simple web form.

## Stage 6

Add AI features slowly.

Possible AI features include reply suggestions, summary creation and customer request classification.

## Long term direction

The long term goal is to turn the best ideas into real portfolio projects that show practical knowledge in AI automation, web development and Python.
