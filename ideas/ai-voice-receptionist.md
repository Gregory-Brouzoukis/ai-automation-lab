# AI Voice Receptionist

This idea is about creating an assistant that can help a business answer basic customer questions and collect appointment requests.

## Problem

Many small businesses receive the same questions every day.

Examples include opening hours, prices, available appointments, location and basic service information.

A simple assistant could help organize these requests and save time.

## Basic idea

The assistant could speak or chat with a customer, ask what they need and collect the important information.

It could then save the request in a simple system such as a spreadsheet or a booking list.

## Possible features

1. Ask for the customer name
2. Ask for phone or email
3. Ask what service they need
4. Ask for preferred day and time
5. Save the request
6. Show a clear summary
7. Help the business owner review requests later

## First simple version

The first version does not need to be advanced.

It can start as a text based assistant that asks questions and saves the answers.

Later it can become a voice assistant.

## Tools that could be used

1. Python
2. Google Sheets
3. Speech recognition tools
4. Text to speech tools
5. A simple web interface
6. AI assistant logic

## Why this project matters

This project connects AI with a real business problem.

It can help me learn how automation works in practical situations and how to build tools that are useful for real users.
